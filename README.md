## Running Locally
**Step 1: Clone the repository using the project's Git URL.
**
`git clone <YOUR_GIT_URL>`

**Step 2: Install the necessary dependencies.
**
`npm i`

**# Step 3: Start frontend
**
`npm run dev`

**# Step 4: Start backend
**
`vercel dev`
```
